About me

Hi, I am a full stack software developer by night and an avid runner, surfer and breakdancer by day. I started with writing simple HTML pages in high school at the informatics course. I quickly became familiar with PHP and MySQL and started programming on more complex dynamic websites on my own. I finished studies at the Faculty of Computer and Information Science at University of Ljubljana.

Around that time PHP era slowly started to fade and I devoted my time to learn about mobile applications and programming in the cloud. By now I have mastered the majority of software development concepts and technologies that are required for zero infrastructure and good application scaling.

I am proficient in Amazon Web Services platform and their services such as EC2, Lambda, DynamoDB, S3, CloudFront, API Gateway, SES and others. One of the projects developed by these technologies exclusively is also this blog. It is running entierly on zero infrastructure higly scalable pay as you go platform using API Gateway, AWS Lambda, DynamoDB, S3 and Cloudfront.

Nowdays I primarily develop software in JavaScript but I can find my way around other languages as well: PHP, SQL, Java, C and Python.

Regarding frontend and UI design I am fluent with JavaScript libraries: AngularJS, JQuery and Sencha Touch for mobile apps, CSS, SASS and HTML5. I also use Cordova/Phonegap for hybrid mobile apps.
